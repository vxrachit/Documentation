# 📘 Semantic Search & QA over Policy Documents

GitHub ¬∑ vxrachit/Semantic-Search-QA-over-Policy-Documents

A FastAPI backend that enables PDF ingestion, vector search using FAISS, and precise Q&A over policy documents with Google Gemini, including citations and per-user isolation.

# Feature Highlights

Feature Role
📄 PDF Ingestion Upload one or more policy PDFs, tied to a specific user_id
🧮 Embeddings Convert document text into dense vectors with sentence-transformers
🔍 Semantic Search Retrieve top-k context snippets using FAISS similarity search
🤖 Gemini QA Generate natural language answers with Google Gemini API
📂 Supabase Storage Store raw PDFs and per-user FAISS indexes
📑 Citations Every answer includes source doc and page reference
👤 User Isolation Each user gets a separate vector index in Supabase

# Dependency Breakdown

# ⚙️ Core API Framework

  • fastapi ‚Üí Defines endpoints (/ingest, /query) and handles request/response lifecycle.
  • uvicorn ‚Üí ASGI server used to run the FastAPI app.
  • starlette ‚Üí Underlying ASGI framework (routing, middleware).
  • anyio, h11, httptools, websockets ‚Üí Async I/O, HTTP, and WebSocket support.

# 📄 PDF Parsing

  • PyMuPDF (fitz) ‚Üí Extracts text from PDFs.

# 🔎 Vector Search & Embeddings

  • faiss-cpu ‚Üí High-performance vector search.
  • sentence-transformers ‚Üí Embedding models (all-MiniLM-L6-v2).
  • transformers, tokenizers, torch ‚Üí HuggingFace stack powering embeddings.
  • safetensors ‚Üí Efficient model weight storage.

# 🤖 LLM Integration (Gemini)

  • google-generativeai ‚Üí Official SDK for Gemini.
  • google-ai-generativelanguage, google-api-core, google-auth, grpcio ‚Üí Google Cloud auth and API stack.

# ☁️ Supabase Integration

  • supabase, supabase_auth, supabase_functions, storage3, postgrest ‚Üí Supabase storage, Postgres, and auth APIs.
  • PyJWT ‚Üí JSON Web Tokens for user isolation.

# 📦 Utilities & Helpers

  • python-dotenv ‚Üí Load .env configs.
  • requests, httpx ‚Üí HTTP requests.
  • joblib, scikit-learn, scipy, numpy ‚Üí Vector operations & preprocessing.
  • tqdm ‚Üí Progress bars during ingestion.
  • regex ‚Üí Text cleaning.
  • watchfiles ‚Üí Live reload for dev.
  • colorama, click ‚Üí Console utilities.

# Setup & Run 

# 1. clone repo
git clone https://github.com/vxrachit/Semantic-Search-QA-over-Policy-Documents.git
cd Semantic-Search-QA-over-Policy-Documents

# 2. create env
python -m venv .venv
source .venv/bin/activate   # Mac/Linux
.venv\Scripts\activate      # Windows

# 3. install dependencies
pip install -r requirements.txt

# Configure .env 

GOOGLE_API_KEY=your_gemini_api_key
GEMINI_MODEL=gemini-2.0-flash
EMBED_MODEL=sentence-transformers/all-MiniLM-L6-v2


SUPABASE_URL=your_supabase_project_url
SUPABASE_SERVICE_ROLE_KEY=your_supabase_key
SUPABASE_BUCKET=policyqa

# Run API 

uvicorn app.main:app --reload

# API Endpoints

# /ingest ‚Üí Upload & embed PDFs

Uploads PDFs ‚Üí extracts text ‚Üí builds embeddings ‚Üí saves FAISS index in Supabase.

curl -X POST "http://127.0.0.1:8000/ingest"   -F "user_id=demo123"   -F "files=@NEP_Final_English_0.pdf"

# /query ‚Üí Ask a question

Retrieves top-k chunks ‚Üí builds Gemini prompt ‚Üí returns citation-backed answer.

{
  "user_id": "demo",
  "question": "What is Education Policy 2020?",
  "top_k": 5
}
